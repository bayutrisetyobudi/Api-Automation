# API Automation with Playwright (JavaScript)

This project uses Playwright with JavaScript (Node.js) to perform automation testing, including API testing.
✅ Conclusion

By following this guide, you can:
Clone the automation project
Install dependencies
Run API tests using Playwright

- Clone the Repository
🚀 Install Playwright
- Install Playwright and required browsers:
npm install @playwright/test

▶️ Running Tests
npx playwright test
